Yemek Öneri Uygulaması (AI Destekli)

Bu proje, kullanıcıdan alınan malzemelere göre yemek önerileri sunan fullstack bir uygulamadır.

Özellikler

- Kullanıcıdan malzeme girişi alınır
- Girilen malzemelere göre tarif önerileri sunulur
- OpenAI entegrasyonu ile dinamik tarif üretimi yapılır
- Mock veri ile de çalışabilen esnek mimari

---

Kullanılan Teknolojiler

### Frontend
- React
- Vite
- CSS

Backend
- Django
- Django REST Framework
- OpenAI API

---

Mimari Yapı

Backend tarafı provider tabanlı bir yapı ile tasarlanmıştır.
